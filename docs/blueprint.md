# **App Name**: IT Mazing

## Core Features:

- Gamified Learning System: Progressive learning through COC 1-4, incorporating lessons, quizzes, and step-by-step progression with score-based advancement and reset conditions.
- Offline Lesson Delivery: Serve text and image-based lessons prior to each quiz, ensuring content is accessible fully offline post initial load using local storage.
- Practice Mode: Enable unlimited practice attempts for each COC with immediate feedback on correct answers and explanations, available offline, to prepare users for official quizzes.
- Achievement and Profile System: Maintain a local player profile including avatar, username, titles, badges, and quiz stats, coupled with an achievement log displaying earned badges and titles with timestamps, persisting data offline.
- Emoji Reaction Pop-ups: Display automatic emoji reactions (Retry, Reset, Pass) with optional sound effects upon quiz completion, providing immediate feedback, fully offline.
- Easter Egg Creator Title: Assign a special 'CREATOR' title and badge if the username matches 'Saint Silver Andre O Cudas', triggering a celebratory emoji on login.  /* This is the creator easter egg */
- 4 Pics 1 Word Mini-Game: Integrate a mini-game where players guess a word based on four images related to the COC topics.  Full completion of the minigame unlocks a special badge.

## Style Guidelines:

- Primary color: Strong Purple (#9D2EC5) to give a sense of creativity.
- Background color: Very light lavender (#F5F0FF).
- Accent color: Violet (#7900FF), for contrast.
- Body and headline font: 'Space Grotesk', sans-serif. A computerized font for headlines and short text chunks, complemented with Inter for longer text.
- Code font: 'Source Code Pro' for displaying code snippets.
- Gaming-style icons for each section of the app.
- Use simple transitions.
